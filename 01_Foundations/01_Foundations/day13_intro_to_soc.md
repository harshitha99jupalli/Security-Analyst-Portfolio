# Day 13 — Introduction to SOC

## What is a SOC?
A SOC (Security Operations Center) is a team that monitors alerts and investigates security incidents in real time.

## SOC Analyst Responsibilities
- Monitor alerts
- Investigate suspicious activity
- Analyze logs
- Decide if alert is true or false positive
- Document findings

## SOC Tier Levels
Tier 1: First level monitoring and triage  
Tier 2: Deep investigation  
Tier 3: Advanced threat hunting  

## SOC Workflow
1. Alert triggered
2. Investigate logs
3. Determine impact
4. Take action
5. Document case
